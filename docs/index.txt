Всім привіт
Поки нема свагера скидую сюди ендпоінти по логіну/реєстрації та користувачам
Базовий URL у Вас буде http://localhost:8050/api/v1

Auth:
POST /register  Реєстрація нового користувача
    req.body: {email, password}
    res: {accessToken}
    accessToken треба зберігати в хедерах зі сторони фронта
    refreshToken зберігається в cookie
POST /login  Логін користувача
    req.body: {email, password}
    res: {accessToken}
    refreshToken автоматично витягується з cookie якщо він там є.
    Якщо ні то створюється
POST /refresh  Оновлення accessToken та refreshToken
    res: accessToken
    
    Перевірка accessToken поки не реалізована. 
    Буду робити через 403 помилку.
    На фронті требу буде робити перевірку на цю помилку.
    Якщо сервер повертатиме 403 то треба викликати refresh

Users:
POST /  Створення нового користувача 
    (Цей route на фронті використовуватися не буде)
GET /  Отримати всіх користувачів
    (Цей route на фронті використовуватися не буде)
GET /:id  Отримати одного
    req.params: {id}
    res: {user}
    Повертає весь об'єкт користувача(можливо навіть з паролем)
PATCH /:id  Оновлення користувача
    req.params: {id}
    req.body: {Object with user data}
    res: {user}
    Повертає весь об'єкт користувача(можливо навіть з паролем)
DELETE /:id  Видалення користувача
    req.params: {id}
    res: none
    За видалення поки не вирішено чи буде чи ні.

Сесії створюються і перевіряються повність на сервері.
